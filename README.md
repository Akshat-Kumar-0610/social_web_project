# social_web_project
Social media website

database : MySQL

working features:
1. authenication (register, login, logout, signin with google)
2. edit profile and profile photo
3. post (both text and images)
4. like, dislike and comments on post
5. follow (news feed model)
6. search


Packages used:
1. pillow
2. openpyxl
3. allauth
4. crispy_forms
